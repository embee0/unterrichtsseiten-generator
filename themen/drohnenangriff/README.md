# Drohnenangriff

Dieses Verzeichnis bündelt alle Dateien zum Thema Drohnenangriff.

- `LERNSEITE.drohnenangriff.md` ist die Hauptquelle der Lernseite
- `drohnenangriff*_aufgabe.py` sind die bearbeitbaren Aufgabenstaende
- `drohnenangriff*.py` sind Demo- oder Musterstaende
- `drohnenangriff_endloesung.py` ist die ausgebaute Endvorschau

Aenderungen am Inhalt sollten zuerst hier passieren und danach mit `python3 build_drohnenangriff_lernseite.py` neu gebaut werden.
